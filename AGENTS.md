# AGENTS.md

## Rol Del Repo

Aquest repo conte Grove Timer, la primera aplicacio de The Code Grove.

## Context Obligatori

Abans de fer canvis:

1. Llegeix `docs/PRODUCT_BRIEF.md`.
2. Llegeix `docs/FEATURE_SPEC.md`.
3. Consulta `../design/grove_timer/GroveTimer Screens.html`.
4. Consulta `../design/docs/DESIGN_SYSTEM.md`.
5. Consulta `../design/tokens/tokens.css`.

## Regles De Producte

- Prioritza una experiencia simple i sense friccio.
- El temporitzador ha de ser llegible a primera vista.
- Els controls principals han d'estar sempre clars.
- No afegir funcionalitats complexes abans de tenir el flux basic solid.
- Les metriques i temps han d'usar estil mono.

## Regles Tecniques

- Si es tria un stack, documenta'l a `docs/AI_CONTEXT.md`.
- Escriu tests quan hi hagi logica de temps, persistencia o estadistiques.
- Evita acoblar logica de timer directament a components visuals si es pot separar clarament.
