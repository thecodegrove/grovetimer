# AI Context

## Que Es Aquest Repo?

La implementacio de Grove Timer. Encara no hi ha stack definit dins aquest repo; el disseny de referencia existeix a `../design/grove_timer/GroveTimer Screens.html`.

## Prioritats

1. Flux basic de timer impecable.
2. UI coherent amb The Code Grove.
3. Logica de temps separada i testeable.
4. Mobile-first.
5. Sense funcionalitats extra abans del MVP.

## Decisions Obertes

- Stack final: web app, PWA, app nativa o hibrida.
- Persistencia local.
- Sons i notificacions.
- Estadistiques.
- Integracio amb media.

## No Fer Sense Confirmar

- Afegir autenticacio.
- Afegir backend.
- Afegir gamificacio complexa.
- Canviar el nom del producte.
- Canviar l'estil fora del sistema `design`.
