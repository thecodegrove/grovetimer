# Feature Spec

## Flux Basic

1. L'usuari obre l'app.
2. Tria o ajusta una durada.
3. Prem iniciar.
4. Visualitza temps restant i progres circular.
5. Pot pausar, reprendre o reiniciar.
6. En acabar, veu un estat final clar.

## Funcionalitats Inicials

- Temporitzador configurable.
- Botons rapids: `+5`, `+10`, `+30`, `+1h`.
- Estat idle, running, paused i completed.
- Progres circular.
- Preferencia de mode clar/fosc si el prototip ho manté.

## Funcionalitats Candidates

- Presets.
- Estadistiques de sessions.
- Integracio amb media o notes.
- Sons suaus de finalitzacio.
- Historial local.

## Logica A Testejar

- Compte enrere.
- Pausa i represa.
- Reinici.
- Finalitzacio exacta.
- Persistencia si s'implementa.

## Accessibilitat

- Controls amb labels.
- Estat anunciat per text, no nomes color.
- Botons prou grans en mobile.
- Contrast suficient en mode fosc.
