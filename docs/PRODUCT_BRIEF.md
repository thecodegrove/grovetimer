# Product Brief

## Producte

Grove Timer.

## Proposit

Un temporitzador bonic i tranquil per ajudar a entrar en una sessio de focus o activitat concreta sense convertir-ho en un sistema de productivitat pesat.

## Usuari

- Persones que volen iniciar sessions de focus amb poca friccio.
- Persones que escolten podcasts, musica o estudien per blocs de temps.
- Usuaris que prefereixen una eina visualment amable i simple.

## Valor

- Comencar rapid.
- Ajustar temps amb botons curts.
- Veure progres de manera clara.
- Mantenir una estetica coherent amb The Code Grove.

## To Del Producte

Calmat, funcional i lleugerament ritual. L'app ha de convidar a començar, no a configurar infinitament.
